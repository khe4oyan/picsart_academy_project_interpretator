# Code syntax

>### Variables type
* num ( numbers )
* str ( text )
* bool ( boolean )
  
```
num age 10;
str name 'Alex';
bool write_interpretator false;
```
___
>### Operators
* if ( condition )
  * ==
  * !=

  * <
* for ( cycle)
* output ( ouput text/number/boolean )

```

}
else;

```
